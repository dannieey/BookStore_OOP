BookStore Management System |

⚙️ Project Description:
This is a console-based BookStore Management System written in Java. The program allows users to interact with a bookstore database where they can browse, add, update, or delete books. The system supports two user roles:

* Admin – manages the bookstore (add, update, delete, and view books).
* User – browses books by genre and can make purchases.

The project is implemented using Java and PostgreSQL for database management.

------------------------------------------------------------

⚒️ How It Works |

Admin Mode:
- Enter "admin" when prompted.
- Choose one of the following actions:
- Add a book (Enter title, author, genre, and price).
- View books (Displays all available books).
- Update book details (Modify an existing book’s information).
- Delete a book (Remove a book from the system).

User Mode:
- Enter "user" when prompted.
- Register by entering a name and surname.
- Select a genre from the available options.
- Choose a book and confirm the purchase.

------------------------------------------------------------

🧑‍💻 Technologies Used |

* Java (JDBC for database connection),
* PostgreSQL (Database for storing books and users),
* Maven (For dependency management).
